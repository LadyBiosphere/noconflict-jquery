What is this? If you have multiple .js libraries in your web project (e.g. WordPress), a conflict may arise when you try to use the “$” symbol. If this happens, it’s because another .js library shares the same symbol and the browser is no longer sure which library you’re trying to use. To prevent such conflicts, you can put jQuery into “noConflict mode”!
